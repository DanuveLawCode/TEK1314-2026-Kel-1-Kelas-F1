# TEK1314-2026-Kel-1-Kelas-F1
Web-Infra Defender (Linux).
Pada skenario Web-Infra Defender (Linux), kami merancang lingkungan lab dengan subnet 192.168.1.0/24 yang terdiri dari tiga node utama: attacker, web server, dan monitoring. Web server menggunakan Ubuntu Server dengan layanan Nginx dan MariaDB, serta menjalankan aplikasi web rentan untuk menguji serangan SQL Injection (SQLi) dan Cross-Site Scripting (XSS). Seluruh aktivitas serangan dari attacker dimonitor menggunakan Security Onion untuk mendeteksi dan menganalisis potensi ancaman sebagai bagian dari pendekatan defensive security.
